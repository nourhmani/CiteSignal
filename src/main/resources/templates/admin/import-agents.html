Importer des Agents Municipaux

Ajoutez plusieurs agents en une seule fois via un fichier CSV


                


                

                    

                        
                        Format du fichier CSV
                    

                    
Votre fichier CSV doit contenir les colonnes suivantes dans cet ordre :

                    
    
                        
  • 
                            
                            nom (obligatoire) - Nom de famille de l'agent
                        
    • 
                        
  • 
                            
                            prenom (obligatoire) - Prénom de l'agent
                        
    • 
                        
  • 
                            
                            email (obligatoire) - Adresse email professionnelle
                        
    • 
                        
  • 
                            
                            telephone (optionnel) - Numéro de téléphone
                        
    • 
                        
  • 
                            
                            adresse (optionnel) - Adresse complète
                        
    • 
                    


                    
Exemple de fichier CSV :

                    
nom,prenom,email,telephone,adresse
                        Dupont,Jean,jean.dupont@ville.fr,0123456789,123 Rue de la Mairie
                        Martin,Marie,marie.martin@ville.fr,0987654321,456 Avenue des Fleurs

                            Points importants à noter
                        
                        
    
                            
  • 
                                
                                Un mot de passe sécurisé sera généré automatiquement pour chaque agent
                            
    • 
                            
  • 
                                
                                Les identifiants (email et mot de passe) seront envoyés par email à chaque agent
                            
    • 
                            
  • 
                                
                                Les agents dont l'email existe déjà dans le système seront ignorés
                            
    • 
                            
  • 
                                
                                Assurez-vous que le fichier utilise l'encodage UTF-8 pour éviter les problèmes d'accents
